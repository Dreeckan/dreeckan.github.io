# Des outils pour se simplifier Git

# VsCode

VsCode fourni déjà une intégration (plutôt efficace et pratique) de Git. Pour avoir un historique assez visuel, j'ai installé un plugin Git History (téléchargeable ici (opens new window) ou dans VsCode)

Pour installer un plugin dans VSCode, aller dans le menu de gauche, partie Extensions (avec 4 petits carrés). Pour chacun de ces plugins, je vous conseille d'en regarder la doc, pour en découvrir toutes les fonctionnalités et les éventuels requis.

# PhpStorm

PhpStorm fourni déjà une intégration très complète de Git (similaire à VsCode avec le plugin Git History) et vous n'avez en général rien de plus à installer.

# GitKraken

⚠️ Outil payant pour les dépôts privés et certaines fonctionnalités ! Voir les tarifs de GitKraken (opens new window), avant toute chose.

Disponible sur le site de GitKraken (opens new window), cet outil multi-plateforme permet de gérer Git de manière visuelle, mais aussi vos tickets ou vos PR, si vous avez un compte payant !
